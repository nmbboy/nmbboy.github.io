搭建hexo的过程

2021-08-19
字数统计: 463   |   阅读时长≈ 1 分钟

其实这次博客的搭建也是一次偶然，也想了想还是应该对自己的一些事情记录一下。

#环境搭建
博客目前为托管在github上的，后期我想让他运行在自己的服务器上，让github作为一个备份。

由于hexo是由Node.js写的，主要是需要以下的环境

  • 一台liunx 服务器（可以为本地）

  • git

  • Node.js

1.安装git

apt install date

apt-get install git

2.安装Node.js和npm

apt-get install nodejs

apt-get install npm

3.验证环境

nodejs –version

npm -v

git -v

4.Hexo的安装

这里我选择的是全局安装

npm install -g hexo-cli

hexo init hexo

到这里呢，hexo的安装就完成了

可以使用

hexo s

访问 http://0.0.0.0:4000 可以验证一下。

#github环境配置

1.首先让git全局记住我的github账号。

git config –global user.name “Name”

git config –global user.email “Email”

2.生成SSH密钥

ssh-keygen -t rsa -C xxx@qq.com

查看一下是否生成

cat .ssh/id_rsa.pub

3.配置github

上github，创建项目

将我刚刚生成的ssh-key加入到github上，这样就可以让他们两个愉快的玩耍了。

验证一下咱们的ssh是否成功的添加到了github上

ssh -T git@github.cn

4.配置Hexo _config.yml

deploy:

type: git

repo: git@github.com:username/username.github.io.git

branch: master

5.部署到github

完成上面的操作后本地就与github连接上了，咱们开始上传部署代码。

hexo g

hexo d

如果需要修改域名访问的话就需要购买域名，并且需要解析一下。

还有一步需要配置，修改github 页面的设置。这里我已经修改好了域名。

至此就以搭建完成了。
